Overview
This project aims to predict the sale price of houses using machine learning techniques. By analyzing historical housing data, the model identifies patterns and relationships between various features and the final sale price.

Key Features
Data Cleaning & Preprocessing: Handles missing values, outliers, and categorical variables

Exploratory Data Analysis (EDA): Visualizes correlations, distributions, and feature importance

Feature Engineering: Creates new features to improve model performance (e.g., total area, age of property)

Multiple Models Tested: Linear Regression, Decision Trees, Random Forest, and XGBoost

Model Evaluation: Uses RMSE, MAE, and R² Score to compare performance

Tech Stack
Python 🐍

Pandas, NumPy (Data manipulation)

Matplotlib, Seaborn (Visualization)

Scikit-learn (Modeling)

Google Colab (Development)

Results
Best performing model: Random Forest Regressor

R² Score: 0.89 on test data

Key predictors: Location, Square Footage, Number of Bedrooms
